## Description
Web made with React, to practice hooks. It shows an interactive number counter.

## Installation
Use the package manager [npm](https://www.npmjs.com/) to install the React Web App.

```bash
npm install
```
## Run App
You can use [npm](https://www.npmjs.com/).
```
npm start
```
